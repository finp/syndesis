Connecting to OData

Open Data Protocol (OData) is a standard for building and consuming REST APIs. An OData service provides services to clients, such as {prodname}, through OData-compliant HTTP requests. A {prodname} integration can obtain entities from an OData service, and can update, create, or delete entities managed by an OData service. To do this, create an OData connection, and add it to an integration.

It is assumed that you are familiar with the OData specification.

Details for connecting to an OData service are in the following topics:

  • [creating-odata-connections_odata]

  • [adding-odata-connections-read_odata]

  • [adding-odata-connections-write_odata]
